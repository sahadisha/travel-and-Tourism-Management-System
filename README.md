# Travel-and-Tourism-Management-System

A dynamic Travel & Tourism Management System developed using Java, JSP, Servlet, JDBC, MySQL, HTML, CSS, and Bootstrap.
This web-based application allows users to explore tour packages, book tours, and enables admins to manage bookings efficiently.
